# Solar Smart Predictive Analysis

Un sistema de estimación de potencial fotovoltaico enfocado en la ciudad de Bogotá, Colombia, utilizando datos geoespaciales y meteorológicos históricos.

## Objetivo del Proyecto
Extraer y analizar datos de radiación solar mediante la **API NASA POWER** para evaluar la viabilidad y el potencial energético de instalaciones de paneles solares en ubicaciones específicas (Latitud: 4.6097, Longitud: -74.0817).

## Tecnologías y Herramientas
* **Lenguaje:** Python
* **Manipulación de Datos:** Pandas, JSON
* **Extracción de Datos:** API NASA POWER (Requests)
* **Control de Versiones:** Git & GitHub

## Próximos Pasos (Pipeline)
- Conexión y extracción de datos vía API.
- Limpieza y transformación del formato JSON a DataFrames estructurados.
- Análisis exploratorio de radiación solar (GHI, DNI).
- Estimación de generación energética mensual.
